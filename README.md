# 📝 Todo List Web App :-

A modern, responsive, and user-friendly Todo List web application built using HTML, CSS, and JavaScript.

## 🚀 Features :-
 
- ➕ Add tasks with date
- ✅ Mark tasks as completed
- ❌ Delete tasks
- 🔍 Filter tasks (All / Pending / Completed)
- 🌙 Clean dark-themed UI
- 📱 Responsive design

## 🛠️ Tech Stack :- 

- HTML
- CSS
- JavaScript

## 💡 How to Use :-
 
1. Enter your task
2. Select a date
3. Click "Add Task"
4. Manage tasks easily
